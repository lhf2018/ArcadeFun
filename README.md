## 📖 项目简介

ArcadeFun 是一个纯娱乐性质的虚拟游戏厅Web应用，包含8款不同类型的迷你游戏。所有游戏使用统一的虚拟游戏币系统，支持杠杆倍率功能，提供高风险高回报的游戏体验。

### 核心理念

- 🎯 **纯娱乐**：所有得分和奖励均为虚拟展示，无实际价值
- 💾 **无存储**：不保存任何用户数据，刷新页面即重置所有状态
- 🎲 **互通性**：所有8款游戏共用同一游戏币池
- ⚡ **杠杆系统**：支持高风险高回报的倍率选择
- 📱 **响应式**：完美支持PC端和移动端

## ✨ 功能特性

### 核心系统
- **游戏币系统**：初始300枚虚拟游戏币，所有游戏通用
- **杠杆倍率**：1x、2x、3x、5x、8x、10x 六档倍率可选
- **重置功能**：一键重置所有游戏状态
- **实时反馈**：Toast提示 + 余额闪烁 + 音效反馈

### 游戏特色
- 🎰 **老虎机** - 经典拉霸，14种符号，多重赔付规则
- 🎡 **幸运轮盘** - 12个扇形区域，随机旋转抽奖
- 🎈 **打气球** - 反应挑战，三种难度可选
- 🎁 **神秘宝箱** - 盲盒抽奖，概率奖励
- 📈 **涨跌猜** - 数字预测，走势图展示
- 🥚 **扭蛋机** - 收集系统，12种扭蛋图鉴
- 🎫 **刮刮乐** - 即开即奖，三连中奖机制
- 🏀 **体育彩票** - 选号摇奖，多重匹配奖励

## 🚀 快速开始

### 环境要求
- Python 3.7 或更高版本
- 现代浏览器（Chrome/Firefox/Safari/Edge）

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/yourusername/ArcadeFun.git
cd ArcadeFun
```

2. **创建虚拟环境（可选但推荐）**
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Mac/Linux
python3 -m venv venv
source venv/bin/activate
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **运行应用**
```bash
python app.py
```

5. **访问游戏**
```
打开浏览器访问：http://localhost:5000
```

## 🎯 游戏规则详解

### 老虎机
- 3个滚轮，每个显示3个符号
- 14种不同符号，各有不同价值
- 中奖组合：三连、特殊组合、双高价值符号等
- 支持杠杆倍率（1x-10x）

### 幸运轮盘
- 12个扇形区域，7种奖励等级
- 奖励分布：200(1格)、100(1格)、50(2格)、20(2格)、10(2格)、5(2格)、0(2格)
- 随机旋转10-15圈后随机停止
- 支持杠杆倍率

### 打气球
- 30秒限时游戏
- 三种难度：简单(2🪙)、普通(3🪙)、困难(5🪙)
- 点击气球获得积分
- 游戏结束后积分自动转换为游戏币

### 神秘宝箱
- 10种奖励等级，概率分布
- 最高奖励500🪙（概率1%）
- 支持杠杆倍率
- 记录最近5次开箱结果

### 涨跌猜
- 数字范围1-100
- 每次变化±1-20
- 猜对获得10🪙基础奖励
- 支持杠杆倍率
- 实时走势图展示

### 扭蛋机
- 12种扭蛋，4种稀有度
- 单抽15🪙，十连抽135🪙
- 十连抽保底稀有以上
- 收集册记录已获得扭蛋
- 每次抽取额外获得5-15🪙（十连抽50-100🪙）

### 刮刮乐
- 3x3网格刮卡
- 35%中奖概率
- 中奖组合：行、列、对角线三连
- 奖励：💎100🪙、⭐50🪙、🍒30🪙、🍋20🪙
- 支持杠杆倍率

### 体育彩票
- 1-20选号
- 摇出3个球（可重复）
- 匹配规则：3个全中200🪙、中2个50🪙、中1个10🪙
- 支持杠杆倍率

## 🔧 常见问题

### Q: 刷新页面后游戏币重置？
A: 是的，这是设计特性。所有数据仅存储在内存中，刷新即重置。

### Q: 如何获得更多游戏币？
A: 通过玩各种游戏赢取，或点击"重置"按钮恢复到300枚。

### Q: 杠杆功能有什么用？
A: 杠杆会放大消耗和奖励。例如选择10倍杠杆，消耗10倍，奖励也10倍。

### Q: 扭蛋收集册会保存吗？
A: 不会，刷新页面后收集册会清空，点击重置按钮也会清空。

### Q: 可以添加新游戏吗？
A: 可以，按照现有游戏模式在 `app.py` 中添加API，在 `index.html` 中添加页面即可。
